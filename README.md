
# hikurangi-takapau

Learn about Te Takapau a Maui

## how

### install

```
git clone https://github.com/rena-deane/hikurangi-takapau
cd hikurangi-takapau
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
